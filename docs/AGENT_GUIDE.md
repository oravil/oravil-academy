# Lesson Philosophy

One Lesson = One Promise.

Each lesson answers one primary question only.

Never teach future lessons.

Future lessons own their knowledge.

Mention future topics only when required.

Do not explain them.

---

# Writing Style

Write as a senior consultant mentoring a newly hired employee.

Avoid:

- blog writing
- academic writing
- marketing language
- motivational language
- filler

Prioritize:

- clarity
- professional thinking
- business context
- learner application
