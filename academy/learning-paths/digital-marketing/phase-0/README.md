# Phase 0 — Foundations

## Purpose

Establish the conceptual foundation required to operate within any digital marketing context, covering the marketing landscape, media models, the customer journey, and the role of digital channels in business growth.

---

## Skills Acquired

- Distinguish between owned, earned, and paid media.
- Map the stages of the marketing funnel.
- Identify the components of a customer journey.
- Define target audiences using persona frameworks.
- Navigate common digital marketing tools and platforms at an orientation level.
- Understand basic web infrastructure relevant to marketing (domains, hosting, CMS).

- A completed persona profile for a defined target audience.
- A mapped customer journey for that persona across funnel stages.
