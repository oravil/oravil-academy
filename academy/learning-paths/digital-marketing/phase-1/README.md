# Phase 1 — Content and SEO

## Purpose

Develop practical competency in organic visibility through content strategy and search engine optimization, covering keyword research, on-page and technical SEO fundamentals, and content planning aligned to audience intent.

---

## Skills Acquired

- Conduct keyword research using standard tools.
- Classify keywords by search intent.
- Apply on-page SEO principles to web content.
- Identify technical SEO issues at a foundational level.
- Develop a content plan aligned to funnel stages and audience intent.
- Understand off-page SEO and link signals.
- Measure organic performance using web analytics.

- A keyword research report for an assigned topic or brand.
- A content plan covering at least three funnel stages.
- An on-page SEO audit of an existing web page.

---

## Exit Criteria
